Tesla's Products

Model S

Model S Plaid has the quickest acceleration of any vehicle in production. Updated great battery architecture for all Model S trims enables back-to-back track runs without performance degradation.

Model 3

Model 3 comes with dual motor all-wheel drive, 20 Uber turbine Wheels and Performance Brakes for total control in all weather conditions. Model 3 can accelerate from 0-60 mph* in as little as 3.1 seconds.

Model X

With the most power and quickest acceleration of any SUV, Model X Plaid is the highest performing SUV ever built. All Model X powertrains, with updated battery architecture, can deliver instant torque at any speed.

Model Y

Model Y provides maximum versatility and the ability to carry 7 passengers and their cargo. Each second row seat folds flat independently, creating flexible storage for skis, furniture, luggage and more.
